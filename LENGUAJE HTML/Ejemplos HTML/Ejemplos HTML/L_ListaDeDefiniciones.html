Glosario de términos

HTML
HyperText Markup Language. Lenguaje que permite diseñar páginas con diversos formatos utilizando etiquetas.
CGI
Common Gateway Interface. Interfaz que permite invocar programas escritos en cualquier lenguaje interpretado o compilado que esté disponible en el servidor donde residen.
WWW
World Wide Web.
Aplicación basada en el modelo cliente-servidor. Permite a un cliente obtener páginas HTML, que residen en un servidor remoto, que contienen información en forma de texto, dibujos, gráficos, sonido y vídeo y que pueden tener referencias a otras páginas HTML.
